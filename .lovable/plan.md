## Что делаем в Lovable (это я)

### 1. Убираем авторизацию
- Удаляю `src/routes/auth.tsx` (страница входа/регистрации)
- Убираю все упоминания Supabase Auth из кода

### 2. Убираем Storage и историю примерок
- Перевожу AI-примерку на полностью in-memory работу:
  - Пользователь выбирает фото в браузере → отправляется как base64 на сервер → AI возвращает результат → показываем в браузере

- Убираю обращения к bucket `tryon` из `src/components/lana/TryOnSection.tsx`
- Удаляю таблицу `tryon_results` (миграция)
- Удаляю таблицу `lead_requests` (она уже не используется — заявки идут письмом)
- Bucket `tryon` удалим после переезда (или сразу, по желанию)

### 3. Готовлю код к запуску на обычном Node.js сервере (Timeweb VPS)
- Перевожу сборку с Cloudflare Workers на Node.js (это штатный режим TanStack Start)

- Проверяю, что Resend и OpenRouter работают из Node.js (они и так работают через обычный `fetch`)
- Заменяю чтение переменных окружения: вместо `process.env.RESEND_API_KEY_DIRECT` → `process.env.RESEND_API_KEY` (стандартное имя для своего сервера)

### 4. Подготовка к деплою
- Пишу `README-DEPLOY.md` с пошаговой инструкцией для Timeweb:
  - Какой тариф VPS взять

- Готовлю файл `.env.example` со списком нужных переменных

## Что вы делаете параллельно

1. **Покупаете домен** (например `lanastone.ru`) — у reg.ru или прямо в Timeweb
2. **Регистрируетесь на Timeweb Cloud** и берёте VPS:
   - Тариф: **Cloud Server**, 2 vCPU / 2 ГБ RAM / 30 ГБ NVMe
   - ОС: Ubuntu 22.04 LTS
   - Локация: Москва или СПб
   - Ориентировочно 300–500 ₽/мес
3. Когда домен и сервер будут готовы — я даю инструкцию по развёртыванию
